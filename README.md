# Diwali_Sales_Analysis
This project is based on EDA process .Through this project we can easily understand EDA process and this project is smade in jupyter notebook.
🪔 Diwali Sales Analysis
📌 Project Overview

This project analyzes sales data during the Diwali festival to understand customer behavior and buying patterns.
The main goal is to find useful insights that can help businesses improve their sales and marketing strategies.

🎯 Objectives
Identify top customers based on gender, age, and occupation
Analyze which states generate more sales
Find the most popular product categories
Understand customer purchasing behavior
🛠️ Tools & Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
🧹 Data Cleaning

Performed data cleaning to ensure accurate results:

Removed unnecessary columns
Handled missing values
Dropped null records
Converted data types (e.g., Amount column)
📊 Exploratory Data Analysis (EDA)
👩 Gender Analysis
Female customers purchased more products than males
Women also spent more money
🎂 Age Group Analysis
Most buyers are in the 26–35 years age group
This group contributes the highest sales
🌍 State-wise Analysis
Top contributing states:
Uttar Pradesh
Maharashtra
Karnataka
💍 Marital Status Analysis
Married customers purchase more
Married women are the top buyers
💼 Occupation Analysis
Major buyers are from:
IT sector
Healthcare
Aviation
🛍️ Product Category Analysis
Most popular categories:
Food
Clothing
Electronics
📦 Top Products
Identified top 10 highest-selling products
📈 Visualizations

Used different charts to represent data:

Bar charts
Count plots

These visualizations help in better understanding of trends and patterns.

✅ Conclusion
Married women aged 26–35 are the primary customers
They mostly belong to Uttar Pradesh, Maharashtra, and Karnataka
They work in IT, Healthcare, and Aviation sectors
Preferred product categories are Food, Clothing, and Electronics

Helps businesses target the right audience
Improves marketing strategies
Increases sales performance
